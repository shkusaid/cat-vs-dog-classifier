# Cat vs Dog Image Classifier

A PyTorch-based convolutional neural network (CNN) that classifies images of cats and dogs. The project includes preprocessing, training, validation, and testing pipelines, and can be trained on custom datasets. GPU acceleration is supported for faster training.

## Features

- Classifies images as cats or dogs using CNN
- Supports GPU acceleration
- Easy to train on custom datasets
- Includes training and testing scripts
